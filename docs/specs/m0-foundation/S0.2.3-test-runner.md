# S0.2.3 — Vitest per package + smoke tests

## Goal
A working test runner in every package, with one trivial passing test each, runnable
repo-wide. Establishes the harness the core determinism test (S0.3.2) will plug into.

## Files
- root `vitest.config.ts` (or per-package configs + workspace)
- `packages/*/src/*.test.ts` (one smoke test per package)
- root `package.json` script: `test` → `vitest run` recursively
- per-package `test` scripts
